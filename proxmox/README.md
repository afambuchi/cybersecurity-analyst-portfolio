# Proxmox Virtual Environment Configuration

This folder contains documentation related to the Proxmox hypervisor that hosts the SOC lab.  
It includes network bridge configuration, VM creation standards, storage layout, and troubleshooting notes.

Proxmox serves as the foundation of the lab and enables:
- Flexible VM provisioning
- Snapshotting and rollback
- Bridge-based network segmentation
- Remote access through the web UI
