# Infrastructure Overview

This folder contains documentation for the underlying infrastructure that supports the SOC lab.  
It covers the virtualization environment, VM inventory, network layout, naming conventions, and general provisioning notes.

The purpose of this documentation is to:
- Provide a reference for all virtual machines and their roles
- Track how networks and bridges are structured in Proxmox
- Maintain consistent naming and configuration practices
- Support future expansion into Windows infrastructure, SIEM, and segmentation

Files included:
- vm-inventory.md
- network-plan.md
- naming-conventions.md
- provisioning-notes.md
