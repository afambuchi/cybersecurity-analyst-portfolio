# Goals and Roadmap

This lab is being developed in phases.  
Each phase builds on the previous one to create a comprehensive SOC training and research environment.

## Phase 1: Core Infrastructure (Completed)
- Proxmox installation
- Creation of external and internal networks
- Custom Ubuntu firewall with NAT and filtering
- Analyst VM with network connectivity validated

## Phase 2: Identity and Endpoints (Next)
- Windows Server for Active Directory
- Windows 11 client
- Sysmon configuration
- Group Policy hardening

## Phase 3: Monitoring and Visibility
- Wazuh SIEM deployment
- FleetDM for endpoint visibility
- Log forwarding and parsing
- Basic alert development

## Phase 4: Segmentation and Security Architecture
- VLAN design
- Dedicated zones for endpoints, infrastructure, and tools
- Updated firewall rules and routing

## Phase 5: Detection Engineering and Automation
- Python-based tools for enrichment and triage
- Custom detection rules
- Investigation exercises
- Integration with SIEM workflows

This roadmap will evolve as new needs or teaching opportunities arise.
